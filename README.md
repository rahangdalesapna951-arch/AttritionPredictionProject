
# Employee Attrition Prediction Dashboard

## Overview
This project predicts employee attrition (whether an employee is likely to leave the company) using machine learning techniques. It analyzes employee data and provides insights through an interactive dashboard.

## Objective
The main objective of this project is to help organizations understand employee attrition patterns and support HR decision-making using predictive analytics.

## Technologies Used
- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Project Features
- Data preprocessing
- Exploratory data analysis (EDA)
- Machine learning model for prediction
- Interactive Streamlit dashboard
- Visual insights and recommendations

## How to Run
1. Install required libraries:
   `pip install -r requirements.txt`

2. Run the project:
   `streamlit run app.py`

## Files Included
- app.py
- Jupyter notebooks
- model (.pkl files)
- dataset
- charts and reports

## Conclusion
This project demonstrates how machine learning can be used to predict employee attrition and provide actionable insights for HR management.
